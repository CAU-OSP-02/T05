## Description

손을 인식하여 진행되는 리듬게임 구현.

화면에 제시된 문제를 보고 손을 문제에 제시된 실루엣에 일치하도록 움직여 문제를 통과하는 리듬게임이다.
시간 안에 문제를 해결하여 많은 점수를 얻는 것을 목표로 한다.
